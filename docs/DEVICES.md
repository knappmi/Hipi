# Device Control

## Unified Device Manager
- Central manager for all device types
- Abstract interface for device control
- Device registry for discovered devices

## MQTT Integration
- Connect to MQTT broker
- Discover MQTT devices
- Control devices via MQTT

## WiFi Device Discovery
- TP-Link Kasa devices
- Philips Hue devices
- Automatic device discovery

## Device Registry
- Centralized device storage
- Device metadata and capabilities
- Device state tracking

**API**: `/api/v1/devices/*`

See device control system documentation for details.
