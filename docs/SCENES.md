# Scenes & Automation

## Scene Management
- Create scenes with multiple device states
- Activate scenes via API or voice
- Update and delete scenes
- Scene metadata (icons, descriptions)

## Enhanced Automation
- Scene automations (activate scenes automatically)
- Time-based triggers
- Event-based automations
- Conditional automations

## Voice Commands
- "Activate movie night"
- "Activate bedtime scene"
- "Turn on away mode"

**API**: `/api/v1/scenes/*`

See [AUTOMATION_SCENES_SYSTEM.md](../AUTOMATION_SCENES_SYSTEM.md) for details.
