# Media Control

## Device Management
- Register TVs, speakers, and media devices
- Device capabilities tracking
- Current state management

## Playback Control
- Play, pause, stop
- Volume control (0-100)
- Track navigation (next/previous)
- Session management

## Platform Integrations
- **Spotify**: Search and play tracks (OAuth ready)
- **YouTube**: Search and play videos (API key ready)
- **Local Media**: Local media playback support

## Voice Commands
- "Play music"
- "Pause"
- "Volume up"
- "Next track"

**API**: `/api/v1/media/*`

See [MEDIA_CONTROL_SYSTEM.md](../MEDIA_CONTROL_SYSTEM.md) for details.
